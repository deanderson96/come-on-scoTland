About the site

About

A Scotland-first World Cup microsite.

This static website is designed for GitHub Pages and built with plain HTML, CSS and JavaScript. It attempts to enhance the experience with public football data from TheSportsDB while keeping a complete fallback experience for reliability.

The 2026 FIFA World Cup is staged across the United States, Canada and Mexico. This site keeps the focus on Scotland: fixtures, player cards, group context and supporter-friendly updates.

No backend Runs as a static site from GitHub Pages.
Resilient data Live API attempts with graceful fallback rendering.
Responsive Mobile-first cards, tables and navigation.
Accessible Semantic landmarks, skip link, live regions and keyboard-friendly controls.
