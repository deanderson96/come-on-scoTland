About the hub

Built for the Tartan Army, ready for GitHub.

This static website is designed as a Scotland-focused World Cup 2026 microsite. It combines a retro football programme look with modern responsive layouts, accessible markup and lightweight JavaScript.

The 2026 tournament is hosted across the USA, Canada and Mexico. This site avoids making unsupported selection or squad claims by labelling fallback content clearly and preferring live data when the API responds.

Site features

  • No build step or backend
  • TheSportsDB API client
  • Graceful loading, error and fallback states
  • Responsive cards, tables and navigation
  • Editable mock data for quick updates
